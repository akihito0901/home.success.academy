# ホームサクセスアカデミー 公式サイト

現役フリーランス講師が作った、"稼ぐ"ことに特化したWebスキルスクールのLP（ランディングページ）。

## コンセプト
教えるのが本業の先生ではなく、いま実際にWebで稼いでいるフリーランスが作った、稼ぐことに特化したカリキュラム。
主婦・子育て中の方／副業したい会社員向け。「何から始めれば…」「スクール高いですよね」「ほんとに意味あるの？」という不安に正面から応える構成。

## 構成
- ヒーロー（メインビジュアル）
- 悩み・共感（3つの不安に共感）
- コンセプト／現役講師紹介
- 学べる4スキル（ライティング / デザイン / SNS運用 / ネット基礎）
- 選ばれる理由
- 学びの流れ（4ステップ）
- 料金プラン（3プラン）
- FAQ
- CTA（無料説明会）

## ファイル
```
index.html          … LP本体（CSSはインライン、1ファイル完結）
images/             … 各セクションの画像を配置
  hero.jpg          … ① ヒーロー
  instructor.jpg    … ③ 講師
README.md
```

## 画像について
画像はChatGPTで生成して `images/` に配置してください（プロンプトは別途「HP画像生成プロンプト.md」参照）。
画像未配置でもプレースホルダ枠が表示されるので、レイアウトは崩れません。
